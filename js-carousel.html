C-A-R-O-U-S-E-L

Карусель — это динамический элемент пользовательского интерфейса, который автоматически переключает серию изображений или слайдов контента.
Нажатие на стрелки справа\слева сдвигают следующую\предыдущую группу слайдов. Нажатие на навигацию внизу переключает на соответствующую группу слайдов. На устройствах с поддержкой прикосновения на контенте и пролистывания по горизонтали сдвигают группу слайдов.

Подробнее

Адаптивность

Загрузка страницы, при разной ширине экрана просмотра, скрипт учитывает и перестраивает количество отображаемых (видных на экране) элементов-карусели.

Гибкость

  1. Предусматривает возможность не создавать или перестраивать структуру html-разметки для скрипта карусели, а встраивать в уже существующую разметку (тег-контенер содержит теги которые планируется сделать элементами карусели, просто добавляется несколько атрибутов в тег-контейнер)
  2. Элементы группы контента могут быть не только картинки, но и почти любой тег с настраиваемым селектором класса

Реализация

Рендеринг (отрисовка) каркаса карусели происходит через js.
Настройка состоит из добавления тегам нескольких пользовательских атрибутов + при инициализации добавления объекта с назначением количества отображаемых элементов для разной ширины экрана просмотра carousel({'769':3,'479':2,'0':1}) , где ключ соответствует медиа запросу min-width: 769px, а значение — количество отображаемых элементов-карусели.
Настройка (пользовательские атрибуты):

  1. data-carousel="carousel-1" — добавляется тегу-контейнеру внутри которого элементы-карусели, значение атрибута равно классу, который на этапе рендеренга назначится тегу-контейнеру содержащему карусель, это способствует гибкой настройке определенного этим уникальным классом экземпляру карусели.
  2. data-carousel-elm="carousel__content" — добавляется тегу-контейнеру, значение это класс элементов-карусели

Добавление класса carousel__act элементу-карусели делает показ карусели с него.
