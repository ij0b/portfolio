Модальное окно — окно, которое появляется на сайте поверх других документов и окон и делает весь находящийся под ним контент недоступным до тех пор, пока вы с ним не провзаимодействуете.
Нажатие на ссылку (текст\картинка) отображает соответствующий ей контент в всплывающем модальном окне.

Подробнее

Гибкость

  1. Предусматривает возможность не создавать или перестраивать структуру html-разметки для скрипта модального окна, а встраивать в уже существующую разметку
  2. Добавлять атрибуты точечно только нужным тегам в узле
  3. Элементы группы контента могут быть не только картинки, но и почти любой тег с настраиваемым селектором класса

Реализация

Рендеринг (отрисовка) каркаса модального окна происходит через js.
Настройка состоит из добавления тегам нескольких пользовательских атрибутов.
Пример №1 (1 элемент) настройка (пользовательские атрибуты):

  1. data-modal-window-a="my-value-1" — добавляется ссылке (текст/картинка/почти любой тег) на контент
  2. data-modal-window-content="my-value-1" — добавляется контенту (почти любой тег) который планируется отображать по нажатию на ссылку. Значение атрибута контента = значению атрибута ссылке = "my-value-1"

На этом настройка завершается, это достаточно локанично.

Пример №2/№3 (+1 элементов) настройка (пользовательские атрибуты):

  1. data-modal-window-group="img"/ ".selectorClass" — Добавляется тегу содержащему элементы-ссылки группы и тегу содержащему элементы-контент группы. Значение "img" для группы состоящей из картинок (тег img). Значение ".selectorClass" есть селектор класса, для группы состоящей из почти любых тегов с этим классом.
  2. data-modal-window-a="my-value-3" — добавляется тегу содержащему элементы-ссылки группы. Значение атрибута группы контента = значению атрибута группы ссылок = "my-value-3".
  3. data-modal-window-content="my-value-3" — добавляется тегу содержащему элементы-контент группы. Значение атрибута группы контента = значению атрибута группы ссылок = "my-value-3".
  4. data-modal-window-href="img/gif221.gif"/ "1" — внутри тега группы ссылок добавляется ссылке. Значение атрибута элемента группы ссылки = значению атрибута элемента группы контента src тега img = "img/gif221.gif" (или "1" порядковому номеру элемента группы конетнта если значение атрибута data-modal-window-group равно не "img", а селектору класса)

Коротко. Два атрибута "-group" и "-a" добавляется контейнеру-тегу содержащему ссылки "-href", и два атрибута "-group" и "-content" добавляется контейнеру-тегу содержащему элементы-контент группы (или картинки если "-group" = "img" или почти любой тег с селектором "selectorClass", если "-group" = ".selectorClass").
